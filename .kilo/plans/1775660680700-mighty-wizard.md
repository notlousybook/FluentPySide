# FluentPySide Fixes Plan - FluentWinUI3 Compliance

## Project Overview
This plan addresses multiple FluentWinUI3 compliance issues in the FluentPySide project. The main areas are:
- Main gallery app (`examples/gallery.qml`)
- NavigationView (`examples/NavigationView.qml` and `fluentpyside/fluent_winui3/NavigationView.qml`)
- QML controls in `fluentpyside/fluent_winui3/`

## 3. NavigationView Sidebar Issues

### 3.1 Toggle Icon Alignment
**Issue:** Toggle icon not aligned properly when opened/closed.

**Fix:** Adjust toggle button positioning in `NavigationView.qml`:
- Proper horizontal centering when compact (icon only mode)
- Proper left alignment with padding when expanded

### 3.2 Pill Stretch Animation
**Issue:** Selection pill fades in/out instead of stretching/flowing.

**Fix:** In `examples/NavigationView.qml`:
- Change pill `y` animation to `height` stretch animation during transition
- Use `SequentialAnimation` with proper easing for smooth "grow" effect
- Make pill grow past target then settle back during navigation

### 3.3 Overlay Closes Sidebar on Content Click
**Issue:** Clicking anywhere in content area closes sidebar unexpectedly.

**Fix:** Ensure overlay dismiss area has proper z-ordering and visibility conditions:
- Only dismiss when in overlay mode (small screens)
- Check that clicks in content area don't propagate to dismiss handler

### 3.4 Text Disappearing
**Issue:** Sidebar text gone - likely visibility/opacity animation timing.

**Fix:** Check text opacity animations - should fade on compact, not disappear instantly.

### Files to Modify
- `examples/NavigationView.qml` (toggle button section, pill animation, overlay)
- `fluentpyside/fluent_winui3/NavigationView.qml`
