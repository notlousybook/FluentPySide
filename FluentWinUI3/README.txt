This folder contains the FluentWinUI3 QML import tree copied from an upstream PySide6 installation.

It is intended to be distributed as part of the fluentpyside package so projects can import
QtQuick.Controls.FluentWinUI3 without requiring a local PySide6 copy.

Do not include compiled plugin binaries (.dll/.so/.dylib) in the package if you want a
cross-platform, small wheel. Prefer recommending PySide6 or PySide6-Essentials on target systems.
